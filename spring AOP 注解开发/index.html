AOP 操作（AspectJ 注解）

1. 创建类，在类里面定义方法

2. 创建增强类（编写增强逻辑）

3. 进行通知的配置

  • 在 spring 配置文件中，开启注解扫描
  • 使用注解创建 User 和 UserProxy 对象
  • 在增强类上面添加注解 @Aspect
  • 在 spring 配置文件中开启生成代理对象
  • 配置不同类型的通知

在增强类的里面，在作为通知方法上面添加通知类型注解，使用切入点表达式配置

在增强类上面添加注解 @Order(数字类型值)，数字类型值越小优先级越高
