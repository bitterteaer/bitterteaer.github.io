固定整个页面只有100%的高度，不能进行下拉，当组件的内容超过100%时候在个自组件内部加上滚动条

Created2025-03-12|Updated2025-03-12
|Post Views:

要实现这个效果，可以使用 CSS 的 height: 100vh 和 overflow 属性来控制页面和组件的滚动行为。以下是具体的实现步骤：

1. 设置 html 和 body 高度

确保 html 和 body 始终占满整个视口（100% 高度），并且不允许滚动：

2. 让主容器也占满整个页面

然后，为你的主容器（例如 .container）也设置 100% 高度：

3. 让内容超出的组件内部滚动

对于可能溢出的组件（例如 .content），使用 overflow: auto; 使其内部滚动，而不是整个页面滚动：

完整示例

效果说明

  • 页面不会滚动（overflow: hidden; 限制 body ）
  • 组件 content 内部滚动（overflow: auto; 允许超出内容滚动）
  • **固定的 header 和 footer**，不会随着内容滚动
